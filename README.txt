# 简易管理员菜单 v1.2 强制打开补丁

此版本修复了模组启用后，按 F6 或右键点击地面没有反应的问题。

## 更新内容

- 在屏幕左下方附近添加了一个常驻的“管理”按钮。
- 为 F6 添加了多种快捷键监听方式。
- 保留右键点击地面打开菜单的功能。
- 增强控制台日志记录。
- 增加延迟加载保障，进入存档后会自动显示“管理”按钮。

## 打开方法

进入存档后，找到写有以下文字的小按钮：

管理

点击该按钮即可打开或关闭菜单。

也可以使用：

右键点击地面 > 管理员菜单 > 打开管理员菜单

或者按：

F6

## 安装方法

先删除旧的 SimpleAdminMenu 文件夹。

然后将新的 SimpleAdminMenu 文件夹放入：

Windows:
C:\Users\YOURNAME\Zomboid\mods\SimpleAdminMenu

Linux / Steam Deck:
~/Zomboid/mods/SimpleAdminMenu

在《僵尸毁灭工程》的“模组”菜单中启用“简易管理员菜单”。

## 如果仍未显示

打开 console.txt 并搜索：

[SimpleAdminMenu v1.2]

如果找不到该行，说明 Lua 文件没有加载。
如果能够找到，请检查附近的报错信息，以便进一步调整补丁。
